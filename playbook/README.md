# Agent Engineering Playbook

## 🎯 Vision
이 프로젝트는 단순한 "프롬프트 엔지니어링"을 넘어, **소프트웨어 엔지니어링 관점**에서 AI 에이전트를 설계, 구축, 운영하는 방법론을 정립합니다.

## 📂 Structure
* **01_infrastructure**: 에이전트 구동을 위한 하드웨어(GPU) 및 엔진(Ollama) 이해.
* **02_agent_prototypes**: 실제 동작하는 에이전트 코드 및 설계 패턴 (Map-Reduce, Recursive).
* **03_future_designs**: 향후 구현할 고급 기능 및 실험적 아키텍처.

## 🛠️ How to use
이 플레이북들은 단순한 텍스트가 아니라, 향후 **Agent Runner**에 의해 실행 가능한 명세서(Spec)가 되는 것을 목표로 합니다.
각 마크다운 파일의 상단 YAML Frontmatter는 에이전트 실행 설정을 담고 있습니다.
